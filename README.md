<p align = "center" fontsize = 20>
This project is a purchases management system for industries, and manufacturing companies to ease the process of ordering, tracking, and managing oneâ€™s inventory and goods. There is no compelling reason to go to crowded stores or shopping centers during festival seasons. You simply require a PC or a laptop. To access this purchase management system customers just need an email and a password to log in and buy goods. Upon successful login, the customers can browse, purchase, and track their desired products. The application is very simple to navigate and use. 
</p>


### Tools/Technologies Utilized 
- HTML
- CSS
- Java
- Eclipse IDE
- Apache Tomcat Server
- MySQL 8.0 Command Line Client
- Java Servlets
- JSP
- JDBC

<br><br>

<p align = "center" fontsize = 20>
A user can search for a product and even apply filters if needed. A billing and tracking system has also been put in place that can be accessed by the user. The admin has a variety of controls such as adding new products to the database, and tracking and keeping records of all products and transactions. The main objective of this e-commerce project in Java is to provide an online platform for organizations to advertise and sell their products effectively and to attract next-generation clientele. 
</p>
